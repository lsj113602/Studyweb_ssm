跟踪甘特图视图

跟踪甘特图其实是显示上下两行条形图, 用于计划与实际的任务对比
我们将"计划信息"放在Baseline数组中, 里面放的也是任务的一些时间信息
要显示跟踪甘特图, 我们需要满足如下两个条件
1.给任务对象设置Baseline比较基准信息,可以存放多个比较基准信息
2.将甘特图对象设置成跟踪甘特图视图模式:gantt.set('viewMode', 'track');//默认是'gantt'

控制任务树单元格可编辑性

1.不设置列配置的编辑器
2.监听tree的beforecelledit事件, 返回false控制不弹出编辑器编辑
tree.on('beforecelledit', function(e){
    var r = e.record;
    var c = e.column;
    
    if(r.Summary ) {                                        
        if(['Start', 'Finish', 'Duration', 'Work'].indexOf(c.dataIndex) != -1){
            return false;
        }
    }
});
